## 1. 项目目标

为“深圳中欣自动化科技有限公司”制作一个全中文 B2B 官网，面向中国工业客户、半导体设备客户、自动化设备客户、工程师和采购人员。

网站核心定位：

> 面向高端装备的精密直驱运动解决方案供应商，覆盖无铁芯直线电机、专用高速直线电机、DDR 直驱旋转电机、管型直线电机与定制高精度平台。

网站需要达成三件事：

1. 建立专业、可信、偏工程型的品牌形象；
2. 清晰展示 IZU、DB、WB、DDR、IZU7、定制平台产品体系与应用场景；
3. 引导用户咨询选型、下载资料、申请图纸、联系工程师。

## 3. 视觉方向：A「精密蓝图」

整体视觉要像高端精密装备公司的工程蓝图，而不是普通产品黄页。

核心感受：

- 克制、专业、洁净、工程感强；
- 大面积白底/浅灰底，配深蓝结构面；
- 使用细线网格、轨迹线、参数坐标、截面感布局强化“精密运动”；
- 产品图为视觉主角，文案短而硬；
- 动效轻，不做粒子、炫光、大红大金、夸张 3D。

设计 token：

- 深海军蓝：`#071A2F`，用于页头、深色 CTA、页脚、关键标题；
- 工业蓝：`#086ACB`，用于按钮、导航 active、重点线条；
- 电光蓝：`#24A8F4`，少量用于轨迹线、数字强调、hover 细节；
- 工程浅灰：`#EDF3F7`，用于分区背景、参数表底色；
- 纯白：`#FFFFFF`，用于主内容留白；
- 正文深灰：`#263341`，用于长文案。

字体策略：

- 中文优先使用系统无衬线：`Noto Sans SC` 可用则优先，其次 `Microsoft YaHei`、`PingFang SC`、系统 sans；
- 数字和参数标签使用更紧凑的英文字体栈：`Arial` / `DIN-like` fallback；
- 不引入外部字体文件，避免版权和加载问题。

页面装饰语言：

- 固定使用“运动轨迹线 / motion rail”作为品牌级视觉元素；
- 参数卡用细线边框和小号数据标签；
- 产品详情页可使用图纸式背景、尺寸图截图、参数表横向滚动。

## 4. 技术架构

技术栈：

- Next.js 14 或以上，App Router；
- TypeScript；
- Tailwind CSS；
- Framer Motion 负责轻量滚动淡入、数据数字动画、hover 微动效；
- 全站中文简体；
- 响应式适配桌面、平板、手机；
- 不做中英文切换；
- 不接真实后端，表单做前端校验与提交成功演示。

页面目标：

- 运动挑战；
- 推荐产品系列；
- 选型提示；
- CTA。

重点行业：

- 半导体检测：高精度定位、高速度扫描、稳定重复运动；
- IC / LED 固晶：高节拍、重复定位、长时间稳定运行；

- 精密检测与测量：低振动、重复定位稳定；
- 电子制造设备：紧凑空间、多运动轴、高节拍。

## 15. 交互与动效

必须实现：

- 页面滚动淡入；
- 产品卡片 hover 轻微上移；
- 首页数据区数字动画；
- 产品分类筛选；

- 顶部导航固定；
- 联系按钮突出；

不实现：

- 复杂 3D；
- 粒子特效；

- 过度炫酷动效。
