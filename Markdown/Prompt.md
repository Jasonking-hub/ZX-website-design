模块 6：行业应用

标题：
服务于高精度、高速度设备行业

行业卡片：
1. 半导体检测
痛点：高精度定位、高速度扫描和稳定重复运动。
解决方案：通过直驱电机和定制平台提升运动稳定性和响应速度。

2. IC / LED 固晶
痛点：节拍快、重复定位要求高、长时间运行稳定性要求高。
解决方案：使用专用无铁芯直线电机和高加速运动方案。

3. 精密贴合
痛点：低速平稳性、速度波动和定位精度影响贴合质量。
解决方案：通过无铁芯直线电机实现平稳直线运动。

4. 高速自动化
痛点：高节拍设备需要更高加速度和更短稳定时间。
解决方案：通过 WB 高加速电机和定制平台满足高动态运动。

5. 精密检测与测量
痛点：运动平稳性、低振动和重复定位稳定性要求高。
解决方案：通过直驱结构减少机械传动误差。

6. 电子制造设备
痛点：设备空间紧凑、运动轴多、节拍要求高。
解决方案：提供紧凑型管型直线电机和定制化直驱部件。
